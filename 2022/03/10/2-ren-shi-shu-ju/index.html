2 认识数据

2.1 数据类型与统计

数据集由数据对象组成。（表）

一个数据对象代表一个实体。（元组）

属性 (attribute) 是一个数据字段，表示数据对象的特征或功能。（列）

属性的类型：（定类、定序、定距、定比）

  • 标称属性，对称二元属性和非对称二元属性（阳性为1，阴性为0）
  • 序数属性（等级、军队排名）
  • 区间标度属性（摄氏温度、日历等，不存在0点，倍数无意义）
  • 比率标度属性（绝对温度、年龄、质量等）

基本统计描述：

  • 中心趋势度量：均值、中位数、众数
  • 分布趋势度量：方差、四分位数极差、盒装图、离群点

2.2 数据可视化

  • 箱线图 (boxplot) ：分析多个属性数据的分布差异性
  • 直方图 (histogram) ：用来分析单个属性在各个区间变化分布；发现特征对类别的区别性
  • 散点图 (scatter plot) ：用来显示两组数据的相关性分布；用来显示两组数据的类型分布（不同颜色表示男女性身高体重分布）

2.3 数据相似性

2.3.1 数据矩阵和相异性矩阵

数据矩阵 (data matrix，对象—属性结构) ：这种数据结构用关系表的形式或 \(n \times p\) 矩阵 (\(n个对象 \times p个属性\)) 存放 n 个数据对象 : \[ \begin{bmatrix} x_{11}&...&x_{1f}&...&x_{1p}\\ ...&...&...&...&...\\ x_{i1}&...&x_{if}&...&x_{ip}\\ ...&...&...&...&...\\ x_{n1}&...&x_{nf}&...&x_{np}\\ \end{bmatrix} \] 相异性矩阵 (dissimilarity matrix，对象-对象结构)：存放 n 个对象两两之间的邻近度（proximity），通常用一个 n \(\times\) n 矩阵表示： \[ \begin{bmatrix} {0}\\ d(2,1)&{0}\\ d(3,1)&d(3,2)&0\\ {\vdots}&{\vdots}&{\vdots}\\ d(n,1)&d(n,2)&{\cdots}&\cdots&0\\ \end{bmatrix} \] 其中 \(d(i,j)\) 是对象 i 和 j 之间的相异性或”差别“的度量，一般非负，越不同越大。

相似性度量：\(sim(i,j) = 1 - d(i,j)\)

2.3.2 标称属性的邻近性度量

  • 标称属性可以取两个或多个状态
  • 方法：简单匹配，m：匹配次数，p：属性总数，\(d(i,j) = \frac{p-m}{p}\)，我们可以通过赋予 m 较大的权重，或者赋给有较多状态的属性的匹配更大的权重来增加 m 的影响。

eg:

id 属性1 属性2 属性3 属性4
1 弹琴 跳高 唱歌 背诗
2 弹琴 跳远 跳舞 读书

\[ d(1,2) = \frac{4-1}{4} \]

2.3.3 二元属性的邻近性度量

  • 二元属性只有两种状态：0 或 1。
  • 方法：所有的二元都被看做具有相同的权重。则我们得到一个两行两列的列联表，其中 q 是对象 i 和 j 都取 1 的属性数， r 是在对象 i 中取 1、在对象 j 中取 0 的属性数，s 是在对象 i 中取 0、在对象 j 中取 1 的属性数，t 是都取 0 的属性数。属性的总数为 p ，其中 \(p=q+r+s+t\)
1 0 sum
1 \(q\) \(r\) \(q+r\)
0 \(s\) \(t\) \(s+t\)
sum \(q+s\) \(r+t\) \(p\)
  • 对称的二元相异性：每个状态都重要，\(d(i,j)=\frac{r+s}{q+r+s+t}\)
  • 非对称的二元相异性：两个都取值 1 的情况（正匹配）被认为比两个都取值 0 的情况（负匹配）更有意义，负匹配数 t 被认为是不重要的，因此在计算时被忽略，\(d(i,j)=\frac{r+s}{q+r+s}\)
  • 非对称的二元相似性，Jaccard系数：\(sim_{Jaccard}(i,j)=\frac{q}{q+r+s}=1-d(i,j)\)

eg: 一个患者记录表，其中 name 是对象标识符，gender 是对称属性，其余都是非对称二元的。

name gender fever cough test-1 test-2 test-3 test-4
Jack M Y N P N N N
Jim M Y Y N N N N
Mary F Y N P N P N

对于非对称属性，值 Y (yes) 和 P (positive) 被设置为 1 ，值 N (no 或 negative) 被设置为 0 。假设对象（患者）之间的距离只基于非对称属性来计算，则三个患者 Jack、Mary、Jim 两两之间的距离如下： \[ d(Jack,Jim) = \frac{1+1}{1+1+1} = 0.67 \] \[ d(Jack,Mary) = \frac{0+1}{2+0+1} = 0.33 \]

\[ d(Jim,Mary) = \frac{1+2}{1+1+2} = 0.75 \]

这些度量显示 Jim 和 Mary 不大可能患类似的疾病，因为他们相异性最高，而 Jack 和 Mary 最可能患类似的疾病。

2.3.4 数值属性的邻近性度量d(Jim,Mary) = = 0.75

  • 闵可夫斯基距离
    • \(d(i,j) = \sqrt[h]{|x_{i1}-x_{j1}|^{h}+|x_{i2}-x_{j2}|^{h}+\cdots+|x_{ip}-x_{jp}|^{h}}\)
    • 性质：
      • 非负性：\(d(i,j) \geq 0\)
      • 同一性：\(d(i,i)=0\)
      • 对称性：\(d(i,j) = d(j,i)\)
      • 三角不等性：\(d(i,j)\leq d(i,k)+d(k,j)\)
  • 曼哈顿距离 \(h=1\) : \(d(i,j) = {|x_{i1}-x_{j1}|+|x_{i2}-x_{j2}|+\cdots+|x_{ip}-x_{jp}|}\)
  • 欧几里得距离 \(h=2\) : \(d(i,j) = \sqrt[]{|x_{i1}-x_{j1}|^{2}+|x_{i2}-x_{j2}|^{2}+\cdots+|x_{ip}-x_{jp}|^{2}}\)
  • 上确界距离 \(h \rightarrow \infty\) : \(d(i,j)=\displaystyle\lim_{h\to\infty}(\sum_{f=1}^{p}|x_{1f}-x_{jf}|^h)^{\frac{1}{h}}=\max_{f}^{p}|x_{if}-x_{jf}|\)

eg:

令\(x_{1} = (1,2), x_{2} = (2,5)\)，则两点间的曼哈顿距离是\(2+3=5\), 欧几里得距离是\(\sqrt[]{2^2+3^2}=3.61\), 上确界距离是\(\max\{2-1, 5-2\}=3\)
