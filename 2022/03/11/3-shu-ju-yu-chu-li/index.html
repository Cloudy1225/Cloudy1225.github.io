3 数据预处理

数据质量：准确性、完整性、一致性、时效性、可信性、可解释性

数据预处理的主要任务：

  • 数据清理
    • 填写缺失值，平滑噪声数据，识别或删除离群点，并解决不一致性来“清理”数据
  • 数据集成
    • 整合多个数据库、多维数据集或文件
  • 数据归约
    • 降维
    • 降数据
    • 数据压缩
  • 数据转换和数据离散化
    • 正常化
    • 生成概念层次结构

3.1 数据清理

现实世界的数据一般是不完整的、有噪声的和不一致的。

3.1.1 缺失值

  • 忽略元组：当类标号缺少时，通常这么做（监督式机器学习中训练集缺乏类标签）。当每个属性缺少值比例比较大时，它的效果非常差

  • 手工填写缺失值：工作量大

  • 自动填写

    • 使用属性的中心度量（如平均值、中位数、众数）填充空缺值

      >>> import numpy as np
>>> from sklearn.impute import SimpleImputer # d
>>> imp = SimpleImputer(missing_values=np.nan, strategy='mean') # 用均值填充
>>> imp.fit([[1, 2], [np.nan, 3], [7, 6]]) # 填充标准
SimpleImputer()
>>> X = [[np.nan, 2], [6, np.nan], [7, 6]] # 待填充数据
>>> print(imp.transform(X))
[[4.          2.        ]
 [6.          3.666...]
 [7.          6.        ]]
>>> imp = SimpleImputer(missing_values=np.nan, strategy='median') # 用中位数填充
>>> imp.fit([[1, 2], [np.nan, 3], [7, 6]])
SimpleImputer(strategy='median')
>>> X = [[np.nan, 2], [6, np.nan], [7, 6]]
>>> print(imp.transform(X))
[[4. 2.]
 [6. 3.]
 [7. 6.]]

  • 最有可能的值：基于诸如回归、贝叶斯公式或决策树推理

3.1.2 噪声数据

原始数据可能存在噪声（被测量的变量的随机误差或方差）、错误或离群的现象，例如属性“年龄”为“-10”，这显然是错误的数据。

  • 离群点分析 (outlier analysis)：可以通过聚类、盒形图等方法检测离群点=数据，进而删除原始数据中的离群点。
  • 回归 (regression)：可以用一个函数拟合数据来光滑数据。线性回归用于拟合两个属性的“最佳”直线，使一个属性可以用来预测另一个属性。多元线性回归是线性回归的补充，涉及的属性多于两个，并将数据拟合到一个曲面上。

3.1.3 数据不一致

数据不一致包括属性中代码不一致或名称不符。例如：年龄和生日不符；某事物曾评级为 “1, 2, 3”，但现在评级为 “A, B, C”，因为不一致的评级代码会导致数据不一致的问题。

  • 计算推理、替换

  • 全局替换

    # 代码实现——空缺值全局替换
>>> import numpy as np
>>> from sklearn.impute import SimpleImputer
>>> imp = SimpleImputer(missing_values=np.nan, strategy='mean')
>>> imp.fit([[1, 2], [np.nan, 3], [7, 6]])
SimpleImputer()
>>> print(imp.transform([[1, 2], [np.nan, 3], [7, 6]]))
[[1. 2.]
 [4. 3.]
 [7. 6.]]

3.2 数据集成

数据集成是将来自多个数据源的数据组合成一个连贯的数据源。小心集成有助于减少结果数据集的冗余和不一致。这有助于提高其后挖掘过程的准确性和速度。

数据库 B

实体识别指识别来自多个数据源的真实世界的实体。上表中，张三和zs、李四和ls为同一实体，因此在数据集成时可用张三和李四表示这两个不同实体，对数据进行集成。

数据冲突指对于同一真实世界的实体，来自不同源的属性值相互冲突。这可能是因为表示、尺度或编码不同。例如上表中数据库 A 与数据库 B 的身高单位不同。

3.2.4 冗余和相关分析

数据集成时经常发生数据冗余的情况，如相同的属性或对象可能在不同的数据库中有不同的名字，或者一个属性可能由另一个或另一组属性“派生”（例如年收入可由月收入派生而成）。通过相关性分析和协方差分析可以检测出冗余的属性。对于离散变量（标称数据），我们使用卡方检验；对于连续变量（数值属性），我们使用相关系数和协方差，它们都评估一个属性的值如何随另一个变化。

3.2.4.1 标称数据的 \({\chi}^2\)检验

对于标称数据，给定两个属性 A 和 B, 可以用卡方检验进行相关性分析，公式如下：

（A 由 c 个不同值，B 由 r 个不同值） \[ {\chi}^2={\sum}\frac{(Observed-Expected)^2}{Expected}\sim{\chi}^2((r-1)(c-1)) \] \({\chi}^2\)越大，变量越有可能是相关的，但相关性并不意味着变量之间存在因果关系。例如：一个城市中病人的数量和小偷的数量是相关的，但二者并无因果关系，与二者有因果联系的第三个变量为人口。

示例：

下棋 不下棋 合计（行）
喜欢看书 250 ( 90 ) 200 ( 360 ) 450
不喜欢看书 50 ( 210 ) 1000 ( 840 ) 1050
合计（列） 300 1200 1500

已知上表中的数据，其中 250、200、50、1000 为观测值 ( Observed) ，通过表格中行合计乘以列合计再除以总数可计算出期望值 (Expected) ：90、360、210、840， \[ Expected(喜欢看书和下棋)=\frac{300 \times 450}{1500} = 90 \]

\[ {\chi}^2 = \frac{(250-90)^2}{90}+\frac{(50-210)^2}{210}+\frac{(200-360)^2}{360}+\frac{(1000-840)^2}{840}=507.93 \]

对于这个 \(2 \times 2\) 的列联表，自由度为 \((2-1) \times (2-1) = 1\)。对于自由度 1， 在 0.001 的置信水平下，拒绝假设的值是 10.828。我们的计算结果远大于该值，故拒绝 A 和 B 独立的假设，并断言二者强相关。

对变量进行相关性分析之后，可将相关的变量进行数据集成。如下表，两者数据表中的属性”是否下棋“和”是否看书“存在相关关系，可以集成为”是否爱好“。

ww 0

3.2.4.2 数值数据的相关系数

皮尔逊Pearson相关系数： \[ r_{p,q} = \frac{\sum(p-\overline p)(q-\overline q)}{(n-1)\sigma_p\sigma_q} =\frac{\sum (pq)-n\overline p \overline q}{(n-1)\sigma_p\sigma_q} \] 其中，\(n\) 为元组的数目，\(p\) 和 \(q\)为各自属性的值，\(\sigma_p\) 和 \(\sigma_q\)为各自的标准差。

\(r \in [-1, -0.75], (-0.75, 0), {0}, (0, 0.75),[0.75, 1]\)，根据 \(r\) 的取值来判断正负相关及其强弱。

3.2.4.3 数值数据的协方差

在概率论与统计学中，协方差和方差是两个类似的度量，用于评估两个属性如何一起变化。 \[ Cov(p,q)=E((p-\overline p)(q-\overline q)) = \frac{\sum_{i=1}^{n}(p-\overline p)(q-\overline q)}{n} =E(A*B)-\overline A * \overline B \]

  • 正相关：\(Cov(p,q)>0\)
  • 负相关：\(Cov(p,q)<0\)
  • 不相关：\(Cov(p,q)=0\)

某些随机变量对的协方差为 0 ，但不是独立的。仅在某种附加条件下（如数据遵守多元正态分布），协方差为 0 意味着独立。

示例：

两只股票 A 和 B 在5天内有以下值：(2, 5), (3, 8), (5, 10), (4, 11), (6, 14)，如果股票都属于同行业，判断它们的价格会一起上升还是下降。 \[ Cov(A,B)=E(A*B)-\overline A * \overline B=42.4-4 \times 9.6=4 \] 因为\(Cov(A,B)>0\)，可以说两只股票会同时上涨。

3.3 数据归约

数据归约技术可以用来得到数据集的归约表示，它小得多，但仍接近于保持原始数据的完整性。也就是说，在归约后的数据集上挖掘将更有效，仍然产生相同（或几乎相同）的分析结果。

3.3.1 降维

需要降维的场景：

  • 数据稀疏，纬度高。
  • 高维数据采用基于规则的分类方法。
  • 采用复杂模型，但是训练集数目较少。
  • 需要可视化。

降维典型方法——PCA主成分分析法

其设法将原来众多具有一定相关性的属性（比如p个属性），重新组合成一组相互无关的综合属性来代替原属性。通常数学上的处理就是将原来p个属性做线性组合，作为新的综合属性。

PCA 中的线性变换等价于坐标变换，变换的目的是使 n 个样本点在新坐标轴 \(y_1\) 上的离散程度（方差）最大，这样变量 \(y_1\) 就代表了原始数据的绝大部分信息，即使忽略 \(y_2\) 也无损大局，从而把两个指标压缩成一个指标。从几何上看，找主成分的问题就是找出 N 维空间中椭球体的主轴问题。从数学上也可以证明，它们分别是相关矩阵的 k 个较大的特征值所对应的特征向量。

输入：\(n\) 个 \(m\) 维 \((n \times m)\) 样本集 \(X = (x_1,x_2,\cdots,x_m)\)，低维空间维数 \(k\) 。

主成分的计算步骤如下：

  1. 对所有样本进行中心化：\(x_i \leftarrow x_i-\frac{1}{n}\sum_{i=1}^{n}x_i\)

  2. 计算样本的协方差矩阵\(X^TX\) \((m \times m)\)

  3. 计算特征值与特征向量

    • 解特征方程\(|\lambda E-X^TX|=0\)，常用雅可比 ( Jacobi ) 法求出特征值，并使其按大小顺序排列，即\(\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_m \geq 0\) 。
    • 分别求出对应于特征值 \(\lambda_i\) 的特征向量 \(e_i(i=1,2,\cdots,m)\)，要求 \(||e_i|| = 1\)，即\(\sum_{j=1}^{m}e_{ij}^2=1\)，其中 \(e_{ij}\) 表示向量 \(e_i\) 的第 \(j\) 个分量。
    • 计算主成分贡献率及累计贡献率。
      • 贡献率的公式：\(f_i=\frac{\lambda_i}{\sum_{i=1}^{m}\lambda_i}\)
      • 累计贡献率：\(\alpha_k=\sum_{i=1}^{k}f_i\)
      • 一般取累计贡献率达 85% ~ 95% 的特征值 \(\lambda_1,\lambda_2,\cdots,\lambda_l\) 所对应的第一、第二、第\(l(l \leq m)\)个主成分。

  4. 计算主成分值

    前 \(k\) 个主成分值 \(z = (Xe_1,Xe_2,\cdots,Xe_k)=(z_1,z_2,\cdots,z_k)\) \((n \times k\ 即\ n\ 个\ k\ 维)\)

与通过保留原属性集的一个子集来减少属性集的大小不同，PCA通过创建一个能替换的、较小的变量集“组合”属性的基本要素。原数据可以投影到较小的集合中。PCA常常能够揭示先前未被察觉的联系，并允许解释不寻常的结果。

示例：

学号 语文 数学 物理 化学 英语 历史
1 84 65 61 72 79 81
2 64 77 77 76 55 70
3 65 67 63 49 57 67
4 74 80 69 75 63 74
5 84 74 70 80 74 82

3.4 数据规范化与数据离散化

数据规范化和离散化是利用函数映射的方法，为给定的属性值更换一个新的表示方法，但是每个旧值与新值仍可以被识别。

3.4.1 数据规范化

规范化数据试图赋予所有属性相等的权重。对于涉及神经网络的分类算法或基于距离度量的分类（如最近邻分类）和聚类，规范化非常有效。规范化可以防止具有较大初始值域的属性与具有较小初始值域的属性（如二元属性）相比权重过大。在没有数据的先验知识时，规范化也是有效的。

在我们的讨论中，令 A 是数值属性，具有 \(n\) 个观测值 \(v_1,v_2,\cdots,v_n\) 。

  • 最小—最大规范化
    • 对数据进行线性变化：\(v_i'=\frac{v_i-min_A}{max_A-min_A}(newmax_A-newmin_A)+newmin_A\)
    • 把 A 的值 \(v_i\) 映射到区间 \([newmin_A,newmax_A]\) 的 \(v’\) 。
  • z —分数规范化
    • \(v_i' = \frac{v_i-\overline A}{\sigma_A}\)
  • 小数定标规范化
    • 移动属性 A 的小数点位置，移动位数依赖于属性 A 的最大值
    • \(v_i' = \frac{v_i}{10^j}\)
    • \(j\) 是使 \(max(|v_i’|)<1\) 的最小整数。例如一组数据的最小值为12000，最大值为98000，那么 \(j\) 取值为5。

3.4.2 数据离散化

由于部分数据挖掘算法只适用于离散数据，因此要将原始数据离散化，需将数值属性的原始值用区间标签或概念标签替换。例如将数值收入离散化为高、中、低三档。

常用的非监督离散方法如下：

  • 等宽法：根据属性的值域来划分，使每个区间的宽度相等。
  • 等频法：根据取值出现的频数将属性的值域划分成小区间，并且要求落在每个区间的样本数目相等。
  • 聚类：利用聚类方法将数据划分到不同的离散类别。
